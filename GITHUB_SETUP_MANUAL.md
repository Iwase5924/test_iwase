# GitHub & Antigravity 連携手順書（完全版）

この手順書は、GitHubアカウントとAntigravity（AIアシスタント）を持っている方が、新しくプロジェクトを立ち上げて両者を連携させるためのステップ・バイ・ステップガイドです。

## 0. 事前準備

作業を始める前に、以下の準備ができているか確認してください。

- **Gitのインストール**: ターミナルで `git --version` を実行して確認できます。
- **GitHubアカウント**: すでに作成済みであること。
- **Antigravity**: このチャット画面が開いていること。

---

## ステップ 1: プロジェクトの初期化

まず、PC上の作業ディレクトリをGitリポジトリとして初期化します。Antigravityに以下のようにお願いしてください。

> 「このディレクトリでGitを初期化して。」

Antigravityが `git init` を実行し、`.git` フォルダが作成されます。

## ステップ 2: ユーザー情報の設定（初回のみ）

Gitで誰が変更を行ったかを記録するために、名前とメールアドレスを設定します。Antigravityに以下を伝えてください。

> 「Gitのユーザー名を『自分の名前』、メールアドレスを『自分のメールアドレス』に設定して。」

## ステップ 3: GitHubリポジトリとの紐付け

GitHub上で新しいリポジトリを作成し、そのURLをコピーしてください。その後、Antigravityに連携を依頼します。

> 「GitHubのリポジトリ `https://github.com/ユーザー名/リポジトリ名.git` と連携して。」

Antigravityが `git remote add origin` を実行し、リモートリポジトリとして登録します。

## ステップ 4: 最初のファイルをコミットしてプッシュする

プロジェクトの基本となるファイル（READMEや.gitignore）を作成し、GitHubにアップロードします。

> 「README.mdと.gitignoreを作成して、GitHubに初回のプッシュをして。」

このとき、ブラウザでのログインやアクセストークンの入力を求められる場合があります。その際は表示に従って認証を行ってください。

## ステップ 5: 連携の確認

GitHubのリポジトリページをリロードして、ファイルがアップロードされていれば連携完了です！

---

## 連携後の使い方（例）

連携が完了した後は、Antigravityに以下のような自然な日本語で指示を出すだけで、開発とGit操作を同時に進められます。

- 「Pythonで数値を計算するスクリプトを書いて、`feat/calc` ブランチにコミットして。」
- 「このプロジェクトの使い方をREADMEに追記して、GitHubにプッシュしておいて。」
- 「バグを修正したから、適切なコミットメッセージで記録して。」

## トラブルシューティング

- **「remote origin already exists」と出る場合**: すでに別のリポジトリが紐付いています。`git remote remove origin` を実行してから再度ステップ3を行ってください。
- **プッシュに失敗する場合**: 認証エラーの可能性があります。GitHubの「Settings > Developer settings > Personal access tokens」からトークンを発行し、パスワードの代わりに使用してください。
